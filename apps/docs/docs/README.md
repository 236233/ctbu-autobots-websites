## 🤝 **参与共建**

我们极度欢迎任何形式的贡献：

- 🐞 提交 Issue 反馈问题

- 🎨 优化文档排版或插图

### **如何贡献？（暂未开放）**

1. **Fork 仓库**：[ctbu-autobots-websites](https://github.com/CTBU-AUTOBOTS/ctbu-autobots-websites)  
2. 在 `docs/` 目录下编辑或新增 `.md` 文件  
3. 提交 **Pull Request**，我们会尽快 Review

> ⚠️ **注意**：请勿上传大于 20MB 的文件（如完整工程包、视频等）。  
> 大文件请托管至网盘，并在文档中提供链接。后续将统一迁移至协会资料库。
