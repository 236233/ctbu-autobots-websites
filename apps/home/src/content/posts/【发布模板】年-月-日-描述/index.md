### 转载文章推荐配置

```yaml
# 情况1：原文有明确版权协议 → 沿用原协议
licenseName: "CC-BY-NC-ND-4.0"

# 情况2：原文无协议/不确定 → 保守声明

# 情况3：已获得作者授权可重新许可 → 按授权填写
licenseName: "CC-BY-4.0"
```

### 原创内容推荐配置

| 你的需求                 | 推荐协议              | 示例配置                      |
| ------------------------ | --------------------- | ----------------------------- |
| 希望内容被广泛传播、引用 | `CC-BY-4.0`           | `licenseName: "CC-BY-4.0"`    |
| 禁止他人商用你的内容     | `CC-BY-NC-4.0`        | `licenseName: "CC-BY-NC-4.0"` |
| 要求衍生作品也开源共享   | `CC-BY-SA-4.0`        | `licenseName: "CC-BY-SA-4.0"` |
| 完全禁止转载/修改        | `All Rights Reserved` | `licenseName: "Unlicensed"`   |
| 放弃版权，贡献公有领域   | `CC0-1.0`             | `licenseName: "CC0-1.0"`      |

### 常用许可协议对照表

| 协议名称                              | 标识符                               | 是否允许商用 | 是否允许修改 | 是否要求署名 | 是否要求相同方式共享 | 适用场景                                 |
